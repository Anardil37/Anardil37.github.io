Depuis le serveur

Si l'ajout ne fonctionne pas en utilisant l'adresse IP de la machine, il faut ajouter une entrée (A ou AAA) dans le dns sur le DC et utiliser le nom d'hôte.

Sélectionner Backup Job puis Linux computer…

 

On choisit Server et Managed by backup server

 

Donner un nom à la tâche

 

On clique sur Add puis Individual Computer et on entre le nom d'hôte de la machine et on clique sur Add, stored et ssh credentials pour les credentials

 

On renseigne un compte utilisateur, on coche les cases et on donne le mot de passe root

 

On choisit ce que l'on veut sauvegarder

 

On choisit dans quel répertoire de backup on veut mettre notre sauvegarde

 

On décoche la case pour ne faire la sauvegarde qu'une seule fois

 

On coche la case pour que la tâche s'exécute à la fin
