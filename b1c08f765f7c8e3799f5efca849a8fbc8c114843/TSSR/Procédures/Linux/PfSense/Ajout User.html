Ajout User

source : https://neptunet.fr/openvpn-pfsense/

 

Dans l'interface de pfSense, aller dans System puis User Manager

Cliquer sur le bouton Add.

Entrer le nom d'utilisateur, un mot de passe et cocher la case créer un certificat d'utilisateur.

Entrer les informations pour la création du certificat.

Valider la création de l'utilisateur.
