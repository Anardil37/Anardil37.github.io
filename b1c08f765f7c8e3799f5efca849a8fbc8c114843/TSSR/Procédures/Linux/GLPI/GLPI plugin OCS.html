GLPI plugin OCS

Télécharger le plugin

https://plugins.glpi-project.org/#/plugin/ocsinventoryng

Le mettre dans la VM par serveur web ou par ftp

Extraire l'archive

tar -xf <nom de l'archive>

Copier le plugin dans la liste des plugins GLPI

mv <nom du dossier> /var/www/html/glpi/plugins

Installer le plugin (icone dossier avec un “+” dessus)

Activer le plugin (passer l'icône rouge en vert)

Ajouter le serveur OCS

Dans OCS, Config puis Server, puis  cocher TRACE_DELETED

Dans GLPI, dans la configuration du plugin, aller dans options d’importation, renseignez le chemin vers votre interface web d’OCS, puis sauvegardez
