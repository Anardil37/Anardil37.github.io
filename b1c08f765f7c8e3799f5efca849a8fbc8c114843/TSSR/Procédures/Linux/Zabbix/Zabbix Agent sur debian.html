Zabbix Agent sur debian

Installation

 

Ajouter le repository.

wget https://repo.zabbix.com/zabbix/5.0/ubuntu/pool/main/z/zabbix-release/zabbix-release_5.0-1%2Bfocal_all.deb

 

Faire un apt update puis installer l'agent.

apt update
apt install -y zabbix-agent

 

Configuration

 

On modifie le fichier de configuration pour indiquer l'ip du serveur zabbix.

nano /etc/zabbix/zabbix_agentd.conf


Server=192.168.240.250
Hostname=debian

 

On redémarre le service, puis on vérifie qu'il est en fonctionnement.

systemctl restart zabbix-agent
systemctl status zabbix-agent
