Shadow Copy

Faire des sauvegardes

Faire un clic droit sur le logo Windows dans la barre des tâches. Cliquer sur Gestion du disque

Faire un clique droit sur le disque souhaité, puis cliquer sur Propriétés

Aller dans l'onglet Clichés instantanés puis sélectionner le volume pour lequel on souhaite faire des sauvegardes. Cliquer sur Activer

  • Créer : Permet de créer un cliché
  • Supprimer : Supprime le cliché sélectionné 
  • Rétablir : Permet de revenir au cliché sélectionné

Restaurer seulement certains fichiers

Faire un clic droit dans le dossier puis cliquer sur Propriétés.

Aller dans l'onglet Versions précédentes puis faire un double clic sur la sauvegarde que l'on souhaite utiliser.

Il suffit ensuite de copier les fichiers que l'on souhaite remettre.
