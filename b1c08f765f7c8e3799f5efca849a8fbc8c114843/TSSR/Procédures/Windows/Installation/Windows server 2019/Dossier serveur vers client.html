Dossier serveur vers client

Il faut commencer par créer un dossier à l'emplacement souhaité.

Il faut ensuite faire un clic droit sur le dossier puis cliquer sur propriétés.

Il faut ensuite cliquer sur l'onglet partage, puis cliquer sur le bouton Partager…

Il faut cliquer sur la petite flèche pour choisir l'utilisateur ou le groupe que l'on souhaite puis cliquer sur ajouter

Il faut ensuite choisir les autorisations que l'on souhaite accorder à cet utilisateur ou au groupe. Il faut ensuite cliquer sur Partager.

Sur le client, ouvrir l'explorateur de fichiers et cliquer dans la barre d'adresse.

Entrer le chemin réseau vers le dossier partagé. Si demandé, entrer le nom d'utilisateur et le mot de passe d'un compte du serveur pour avoir l'accès.

Le dossier s'affiche alors dans l'explorateur.
