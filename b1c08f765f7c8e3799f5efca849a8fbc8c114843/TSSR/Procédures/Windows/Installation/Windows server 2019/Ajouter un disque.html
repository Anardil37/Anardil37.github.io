Ajouter un disque

Clic droit sur la machine virtuelle sur la quelle on souaite ajouter un disque, puis sur Settings…

Choisir Hard Disk dans la liste puis cliquer sur Next

Selectionner SATA puis cliquer sur Next

Selectionner le type de disque souhaité (virtuel ou physique)

Si c'est un disque virtuel, choisir la taille de celui-ci ainsi que si l'on souhaite séparer ou non le disque en plusieurs fichiers

Sur le serveur, clic droit sur l'icône windows dans la barre des tâches puis cliquer sur gestion du disque

Il est alors demandé d'initialiser le disque. Laisser sur Partition GPT pour un disque classique, puis cliquer sur ok
