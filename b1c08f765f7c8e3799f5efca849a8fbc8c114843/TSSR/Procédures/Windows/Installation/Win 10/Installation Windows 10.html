Installation Windows 10

  1. Changer le nom de la machine
  2. Bloquer les mises à jour
  3. Mettre une adresse IP fixe

 

 1. Changer le nom de la machine

 

  • Aller dans les paramètres

  • Aller dans Système

  • Cliquer sur A propos de

  • Cliquer sur Renommer ce PC

  • Cliquer sur Modifier

  • Changer le nom

  • Cliquer sur ok

  • Cliquer sur ok

  • Redémarrer

2. Bloquer les mises à jour

  • Aller dans les paramètres

  • Aller dans Windows Update

  • Cliquer sur Suspendre les mises à jour pendant 7 jours

 

3. Mettre un adresse IP fixe

  • Utiliser ""Windows + R"
  • Tapper ncpa.cpl puis cliquer sur ok

  • Clic droit sur l'interface réseau, puis propriétés

  • Double clic sur Protocole Internet version 4
  • Cliquer sur Utiliser l'adresse IP suivante
  • Entrer l'adresse IP, le masque de sous réseau, la passerelle par défaut et le serveur DNS préféré
