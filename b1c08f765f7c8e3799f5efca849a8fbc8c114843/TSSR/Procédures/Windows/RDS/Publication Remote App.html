Publication Remote App

Dans le gestionnaire de serveur, dans la partie RDS, Cliquer sur la Collection puis dans la partie Programmes RemoteApp, cliquer sur tâches puis sur Publier des programmes RemoteApp.

 

Choisir le programme que l'on souhaite mettre a disposition dans la liste ou cliquer sur Ajouter si le programme souhaité n'apparaît pas dans la liste.

Cliquer sur Suivant, puis sur Publier.

 

 

Sur le client, dans un navigateur internet, aller sur l'adresse https://WIN-RDS.tssr.local/RDWeb

S'identifier avec son compte (penser au TSSR\ ).
