Dossiers documents utilisateur

Créer le partage

Dans le gestionnaire de serveur, aller dans Services de fichiers. Aller dans l'onglet Partages.

Cliquer sur tâches puis sur Nouveau partage…

Sélectionner Partage SMB - Rapide.

Ajouter le Nom du partage et vérifier que les chemins d'accès sont corrects.

Cocher les paramètres que l'on souhaite pour le partage.

Personnaliser les autorisations.

Commencer par désactiver l'héritage, puis cliquer sur Convertir les autorisations héritées.

Enlever les autorisation présentes pour n'avoir que celles-ci.

Ajouter ensuite les autorisations pour les utilisateurs, groupes que l'on souhaite.

Finir le partage.

Dans le profil de l'utilisateur, mettre le chemin vers le dossier que l'on souhaite. 

Le dossier est donc mappé automatiquement pour l'utilisateur.

Mettre en place avec une GPO

Configuration utilisateur → Préférences → Paramètres Windows → Dossiers

Indiquer le chemin d'accès en précisant %username% pour qu'il s'adapte à chaque utilisateur.

Créer ensuite un mappage pour que le dossier soit accessible pour l'utilisateur.
