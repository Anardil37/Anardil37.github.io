Ajouter unité organisation

Dans le gestionnaire de serveur, cliquer sur Outils puis Domaines et approbations Active Directory

Clic droit sur tssr.local puis cliquer sur Gérer

Clic droit sur tssr.local puis cliquer sur Nouveau puis sur Unité d'organisation

Entrer le nom de l'unité d'organisation et cliquer sur OK.
