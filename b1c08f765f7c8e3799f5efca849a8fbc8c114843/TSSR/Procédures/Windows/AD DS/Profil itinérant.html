Profil itinérant

Mettre en place le profil itinérant

Créer un répertoire sur le serveur qui contiendra les fichiers des utilisateurs.

Faire un clic droit sur le dossier puis cliquer sur Propriétés. Cliquer sur l'onglet Partage.

Cliquer sur partage avancé > autorisation.

Enlever tous les utilisateurs et ajouter utilisateurs du domaine.

Faire un clic droit puis cliquer sur Propriétés. Cliquer sur l'onglet Sécurité. Mettre les droit NTFS pour tout le monde.

Dans le gestionnaire de serveur, faire un clic droit sur l'utilisateur puis cliquer sur Propriétés.

Dans l'onglet profil, mettre le chemin d'accès au dossier dans le champ chemin du profil.

\\Win-Srv.tssr.local\Profils_itinérants\%username%
