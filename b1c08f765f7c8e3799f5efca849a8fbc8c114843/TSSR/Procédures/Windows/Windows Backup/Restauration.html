Restauration

Dans le gestionnaire de sauvegarde, on fait un clic droit sur Sauvegarde locale, puis on clique sur Récupérer…

 

On choisit ou se trouve la sauvegarde que l'on souhaite utiliser pour la récupération

 

On choisit la sauvegarde que l'on souhaite restaurer

 

On choisit le type de récupération que l'on veut faire

 

On choisit les éléments à récupérer

 

On choisit les options de la récupération

 

On confirme la récupération 

 

La récupération est lancée
