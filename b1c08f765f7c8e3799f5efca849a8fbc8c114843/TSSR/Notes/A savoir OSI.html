A savoir OSI

couche 1 : physique

Fonction : Représentation des bits sur les supports

PDU : bits (représentation, signal)

Supports : Cuivre (coaxial, paires torsadées) et fibre optique (monomode, multimode)

 

couche 2 : liaison de données

Fonction : Faire communiquer des équipements adjacents qui partage la même politique d'accès au média

PDU : trame

 

couche 3 : réseau

Fonction : Aller d'un périphérique final source à un périphérique final de destination

PDU : paquet

Equipements : routeur

 

couche 4 : transport

Fonction : S'assurer, si l'on veut, que tout à bien été transporté

PDU : segment

 

couche 5 : session

Fonction : Ouvrir une session entre les machine

 

couche 6 : présentation

Fonction : Présenter les informations dans le bon format

 

couche 7 : application
