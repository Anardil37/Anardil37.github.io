Cloud 3

Coûts de développement depuis les dépenses d'investissement (CapEx) liés à la construction et l'entretien d'une infrastructure et d'installations vers les dépenses opérationnelles (OpEx) liés à la location de l'infrastructure en fonction des besoins, qu'il s'agisse de calcul, de stockage, de réseau, etc..

Peut être affecté par de nombreux facteurs:

  • Type de ressource
  • Consommation
  • Maintenance
  • Géographie
  • Type d'abonnement
  • Place de marché Azure

Verrous de ressources

Empêche la suppression ou la modification accidentelle des ressources.

Portail Azure

Console web unifiée qui offre une alternative aux outils en ligne de commande. Permet de gérer l'abonnement Azure à l'aide d'une interface utilisateur graphique.

Azure Cloud Shell

Interpréteur de commandes basé sur un navigateur qui vous permet de créer, configurer et gérer des ressources Azure à l'aide d'un interpréteur de commandes. Azure Cloud Shell prend en charge Azure PowerShell et l'interface de ligne de commande Azure.
