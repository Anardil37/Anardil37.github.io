Wine

Logiciel libre qui permet à des logiciels conçus pour windows de fonctionner dans d'autres environnements. Pour ce faire, il implémente une interface technique de type Windows.

Il n'a pas besoin de l'OS Windows pour fonctionner. Il se différencie donc des émulateurs de machines comme QEMU et Bochs. 

Il est sous licence LGPL, après avoir été sous licence WineHQ, puis X11.

Il fournit les outils de développement pour porter du code source Windows et un chargeur de programmes permettant à de nombreux binaires de fonctionner sans modifications.

Acronyme de Wine Is Not an Emulator, avant 1997 WINdows Emulator.

Première version en Juin 1993.

 

Proton

Logiciel gratuit et open-source qui permet aux logiciels conçus pour Windows de s'exécuter sur d'autres systèmes d'exploitation.

Il est développé par Valve, et basé sur un fork de Wine.

Il comprend plusieurs correctifs et bibliothèques pour améliorer les performances et la compatibilité avec les jeux Windows.

Proton lui même est conçu pour être intégré au client Steam, sous le nom de ‘Steam play’, permettant le fonctionnement transparent des jeux et logiciels Windows dans steam, mais il peut être utilisé comme une application autonome.

Il intègre plusieurs bibliothèques qui améliorent les performances 3D. Les couches de transcription Direct3D vers Vulcan, à savoir D9VK pour Direct3D 9, …

Sorti initialement le 21 aôut 2018.

 

Steam

Dans steam, cliquer sur le menu steam, puis sur paramètres.

Dans les paramètres, cliquer sur Steam play, puis cocher la case Autoriser Steam Play pour tous les titres et choisir la version de proton que l'on souhaite utiliser.

Redémarrer Steam.

Cela permet d'utiliser tous les jeux, même ceux qui ne sont pas dans la liste des applications compatibles.

 

Heroic Game Launcher

Un programme open source qui permet d'accéder aux bibliothèques de jeux Epic, GoG et Amazon prime.

Existe pour Windows, Linux, MacOS, mais aussi disponible sur SteamDeck.

Contient une partie Wine Manager, qui permet de choisir la version de wine/proton que l'on souhaite utiliser.

Permet de gérer la compatibilité pour certaines applications.

Permet également de gérer les paramètres de wine/proton

 

KVM

C'est une machine virtuelle basée sur le noyau. C'est un logiciel que l'on peut installer sur des machines linux physiques pour créer des machines virtuelles.

Une machine virtuelle est une application logicielle qui agit comme un ordinateur indépendant au sein d'un autre ordinateur physique.

Il partage avec la machine physique des ressources telles que les cycles du processeur, la bande passante du réseau et la mémoire.

KVM est un composant du système d'exploitation Linux, qui fournit un support natif pour les machines virtuelles sur linux. Disponible sur les distributions depuis 2007.

Pourquoi KVM ?

La machine virtuelle basée sur le noyau peut transformer toute machine Linux en hyperviseur de matériel nu. Cela permet aux développeurs de mettre à l'échelle l'infrastructure de calcul pour différents systèmes  d'exploitation, sans investir dans de nouveaux matériels.

Conçu pour gérer harmonieusement les applications très exigeantes. Tous les systèmes d'exploitation invités héritent des hautes performances du système d'exploitation hôte.
