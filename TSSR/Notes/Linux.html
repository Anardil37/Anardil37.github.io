Linux

Commandes

fdisk -l → liste les disques et infos

lsblk → liste partitions + type (disk/part) + taille + point montage

blkid → liste des partitions avec UUID, le type et PARTUUID

umount → démontage

cat /etc/fstab → affiche les différentes partitions du disque et les points de montage.

cfdisk → utilitaire pour gérer les partitions

df -h → indique l'espace utilisé par les systèmes de fichier

fsck → analyser et réparer un disque

 

systemd-analyze → voir le temps de démarrage

systemd-analyze blame → voir le temps de démarrage par service

 

stat filename → Afficher un max d'infos sur un fichier

df -T filenamecat filename → Afficher le nom du système de fichier et le point de montage

hexdump -C filename → Afficher les données contenues dans un fichier en hexa ou ascii

du -hs /home/user →  Afficher la taille occupée par le répertoire perso de user

du -h -d 1 /home/user → Afficher la taille des répertoires contenus dans le rép. personnel du user

ls -hl filename → Afficher les infos de base

 

Pour faire une sauvegarde

dossier utilisateur → /home/user

dossier config → /home/user/.config

fichiers système → /etc

profil firefox → .mozilla/firefox

réseaux → /etc/NetworkManager/system-connections

paquets debian installés → dpkg --get-selections > dpkg-list.txt
