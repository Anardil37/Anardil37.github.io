Glossaire

1. **DHCP (Dynamic Host Configuration Protocol)** : C'est un protocole réseau qui permet la configuration automatique des paramètres IP d'une machine ou d'une station sur un réseau informatique¹². Il attribue automatiquement une adresse IP et peut configurer tout à la fois : l'adresse de la passerelle par défaut, celle des serveurs DNS, celle des serveurs NBNS que Microsoft appelle les serveurs WINS¹.

2. **DNS (Domain Name System)** : C'est le système qui traduit les noms de domaine en adresses IP pour que les navigateurs puissent charger les ressources web⁷⁸. Chaque appareil connecté à Internet dispose d'une adresse IP unique que les autres appareils utilisent afin de le trouver⁷.

3. **WSUS (Windows Server Update Service)** : C'est un rôle intégré à Windows Server qui sert à distribuer les mises à jour des produits Microsoft sur les postes de travail et serveurs de votre entreprise⁴⁵. Grâce à WSUS, vous pouvez gérer entièrement la distribution des mises à jour publiées par Microsoft Update sur les ordinateurs de votre réseau⁵.

4. **RDS (Remote Desktop Services)** : C'est un composant de Microsoft Windows qui permet à un utilisateur d'accéder à des applications et des données sur un ordinateur distant, via n'importe quel type de réseau¹¹. L'exécution du système d'exploitation et des applications est réalisée sur un serveur distant via des machines virtuelles¹².

- **Xivo** : Xivo est un logiciel libre de gestion de services d'assistance et de services informatiques. Il permet de passer des appels via votre réseau, traduisant votre voix en signal numérique²².


- **Zabbix** : Zabbix est une solution de supervision professionnelle libre de droit. Il permet de superviser de nombreux paramètres d'un réseau ainsi que la santé et l'intégrité des serveurs, des machines virtuelles, des applications, des services, des bases de données, des sites web, du cloud et plus encore¹.


- **Asterisk** : Asterisk est une solution de téléphonie sur IP, Open Source. Il a été développé par la société Digium et est capable de déclencher des downloads et uploads à l'aide de commandes[^20^].


- **GLPI** : GLPI est un logiciel libre de gestion de services d’assistance (issue tracking system et ServiceDesk) et de services informatiques (ITSM). Il aide les entreprises à visualiser et gérer leurs systèmes d’information de manière complète et efficace¹³.


- **PfSense** : PfSense est un logiciel libre qui fournit des fonctionnalités de pare-feu et de routage pour un réseau⁶.


- **Shadow Copy** : Shadow Copy est une fonctionnalité de Windows qui crée des instantanés (clichés instantanés) des volumes de disque, même lorsqu'ils sont utilisés¹⁶.


- **Samba** : Samba est un genre musical et de danse brésilien²⁷.


- **FTP** : FTP (File Transfer Protocol) est un protocole de communication destiné au partage de fichiers sur un réseau TCP/IP. Il permet, depuis un ordinateur, de copier des fichiers vers un autre ordinateur du réseau, ou encore de supprimer ou de modifier des fichiers sur cet ordinateur[^10^]..

 

**SMB (Server Message Block)** est un protocole client-serveur qui permet d'accéder à des ressources via le réseau, en particulier l'accès à des fichiers, des répertoires complets et d'autres ressources du réseau comme les imprimantes, les routeurs ou les interfaces partagées dans le réseau¹². Il a été développé par IBM en 1983 et est principalement utilisé dans les systèmes d'exploitation Windows¹. Le projet logiciel gratuit Samba offre une solution permettant l'utilisation de Server Message Block dans les distributions Linux et Unix, permettant ainsi une communication multiplateformes via SMB¹.
