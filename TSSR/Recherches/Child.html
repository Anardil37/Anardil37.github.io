Qu’est-ce que le domaine enfant

Un domaine d’arborescence ou une arborescence de domaines est formé selon la structure de domaine parent-enfant. Lorsqu’un domaine est utilisé comme domaine spécifique, ce domaine est appelé domaine enfant. De plus, le domaine principal est le domaine parent.

Un domaine enfant dans Active Directory est un sous-domaine ajouté à un domaine existant dans une forêt Active Directory. Par exemple, si vous avez un domaine `tssr.local` et que vous ajoutez le domaine enfant `tours.tssr.local`, une relation de confiance est automatiquement établie entre ces deux domaines.

Lorsque vous ajoutez un nouveau domaine à une forêt existante, une relation d'approbation est automatiquement créée entre le domaine enfant et le domaine parent. Cette relation d'approbation est transitive et bidirectionnelle.

Pour installer un nouveau domaine enfant, vous devez être connecté en tant que membre du groupe Administrateurs de l'entreprise. Vous pouvez ajouter des domaines enfants à une forêt Windows Server existante à l'aide du Gestionnaire de serveur ou de Windows PowerShell.
