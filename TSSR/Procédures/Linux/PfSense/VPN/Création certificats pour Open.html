Création certificats pour OpenVPN

Source : https://neptunet.fr/openvpn-pfsense/

 

Dans l'interface web, aller ans VPN, puis OpenVPN

Cliquer sur Wizards

Dans type de serveur, choisir Local User Access

On entre les informations nécessaires, puis on clique sur Add new CA

Maintenant, on créer un certificat pour le serveur VPN. Il faut laisser les champs préremplis selon les informations précédentes

Ajouter juste une description puis cliquer sur Create new Certificate.

Il faut vérifier que l'interface sélectionnée soit l'interface WAN.

Sur la même page, on sélectionne l'algorithme de chiffrement, ainsi que l'algorithme de hachage que l'on souhaite.

Toujours sur la même page, on Ajoute une adresse de réseau que l'on souhaite utiliser pour le tunnel, puis on ajoute l'adresse de notre réseau en host only.

Continuer, puis arrivé à l'étape 10, cocher les deux cases tout en bas pour créer automatiquement les règles dans le pare-feu.
