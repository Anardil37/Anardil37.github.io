Exporter la configuration OpenVPN

source : https://neptunet.fr/openvpn-pfsense/

 

Dans l'interface de pfSense, aller dans System puis Package Manager.

Rechercher le paquet openvpn-client-export, puis cliquer sur Install, puis sur Confirm.

Attendre jusqu'à ce que l'installation se termine.

 

Maintenant, retrourner dans le menu VPN, puis OpenVPN et cliquer sur Client Export.

Descendre tout en bas de la page, jusqu'à la catégorie OpenVPN Clients.

On peut maintenant faire un export de la configuration pour l'utilisateur que l'on souhaite.
