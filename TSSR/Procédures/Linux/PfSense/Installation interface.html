Installation interface

Pour accéder à l'interface web de pfSense : http(s)://<ip lan>/

Identifiants par défaut:

  • admin
  • pfsense

A l'étape 2, modifier le hostname si l'on souhaite et le domaine que l'on utilise

A l'étape 3, changer le fuseau horaire

A l'étape suivante, modifier si l'on souhaite l'interface en WAN pour mettre une IP fixe pour qu'elle reste la même

Décocher les 2 cases en bas de la page

A l'étape 6, modifier le mot de passe administrateur de l'interface

Finir l'installation sans rien changer.
