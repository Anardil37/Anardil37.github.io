Installation GLPI

Installation LAMP

Mettre à jour la distribution.

apt update && apt upgrade

Installer les paquets apache2, mysql et php.

apt install apache2 mariadb-server php

Installer les extensions php.

apt install php-xml php-common php-json php-mysql php-mbstring php-curl php-gd php-intl php-zip php-bz2 php-imap php-apcu
apt install php-ldap

Se connecter au serveur mysql.

mysql -u root -p

Créer une base de données pour GLPI et créer un utilisateur en lui donnant les privilèges sur la base de données.

CREATE DATABASE glpibdd;
GRANT ALL PRIVILEGES ON glpibdd.* TO glpiuser@localhost IDENTIFIED BY "glpipass";
FLUSH PRIVILEGES;
EXIT;

Récupérer les fichiers de GLPI.

cd /tmp
wget https://github.com/glpi-project/glpi/releases/download/10.0.10/glpi-10.0.10.tgz

Extraire les fichiers de l'archive et les copier dans le dossier où se situent les sites de apache2.

tar -xzvf glpi-10.0.10.tgz -C /var/www/html

Mettre www-data comme propriétaire du dossier.

chown www-data:www-data /var/www/html/ -R

Modifier les droits du dossier.

chmod 755 -R /var/www/html

 

Configuration GLPI

Choisir la langue puis cliquer sur OK.

Cliquer sur installer.

Vérifier que tout est fonctionnel, sinon, installer ce qu'il faut.

Indiquer le serveur SQL, l'utilisateur et le mot de passe.

Une fois que la connexion est faite, sélectionner la base de données à utiliser.

Décocher la case.

L'installation est terminée. Penser à se rappeler des identifiants.
