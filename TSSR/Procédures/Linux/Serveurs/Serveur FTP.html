Serveur FTP

Installer proftpd

apt install proftpd

Configurer le serveur ftp

nano /etc/proftpd/proftpd.conf

DefaultRoot ~


PassivePorts 49152 65534

Redémarrer le service

systemctl restart proftpd

Créer un utilisateur pour le ftp

useradd -m -s /bin/bash ftpuser
passwd ftpuser
mkdir /home/ftpuser/upload
mkdir /home/ftpuser/download
chown -R ftpuser:ftpuser /home/ftpuser

chmod 755 /home/ftpuser/upload

chmod 755 /home/ftpuser/download

Pour tester

ftp localhost
