Installer et configurer serveur DHCP

Installation

Sur le serveur, aller dans Gérer puis cliquer sur Ajouter des rôles et des fonctionnalités

Choisir Installation basée sur un rôle ou une fonctionnalité puis cliquer sur suivant

Sélectionner le serveur dans la liste puis cliquer sur suivant

Chercher dans la liste Serveur DHCP et cocher la case correspondante

Cliquer sur suivant puis installer

 

 

Configuration

Cliquer sur le drapeau en haut à droite dans le gestionnaire de serveur, puis cliquer sur terminer la configuration DHCP

Cliquer sur suivant puis fermer

Etendue DHCP

Dans le gestionnaire de serveur cliquer sur Outils puis cliquer sur DHCP

Dans le panneau de gauche, cliquer sur IPv4 puis dans le panneau de droite, cliquer sur Autres actions puis Nouvelle étendue…

Entrer le nom de l'étendue puis cliquer sur Suivant

Entrer la plage d'adresse IP que l'on souhaite utiliser puis cliquer sur suivant

Cliquer sur Oui, je veux configurer ces options maintenant, puis cliquer sur suivant jusqu'à arriver à la configuration des paramètres DHCP

Ajouter l'adresse IP de la passerelle par défaut, puis cliquer sur Ajouter, puis cliquer sur suivant

Cliquer sur suivant jusqu'à arriver à la fin puis cliquer sur Terminer
