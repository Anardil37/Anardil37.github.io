Installer et configurer serveur DNS

Installation

Sur le serveur, aller dans Gérer puis cliquer sur Ajouter des rôles et des fonctionnalités

Chercher Serveur DNS dans la liste puis cliquer sur Suivant

Cliquer sur Suivant jusqu'à la fin puis sur Installer

Configuration

Dans le gestionnaire de serveur, cliquer sur Outils puis DNS

Cliquer sur le serveur puis Zones de recherche inversées. Faire un clic droit sur zones de recherche inversées puis cliquer sur nouvelle zone

Choisir zone principale

Choisir Zone de recherche inversée IPv4

Entrer l'adresse IP du réseau

Cliquer sur le serveur puis Zones de recherche directes. Faire un clic droit sur zones de recherche directes puis cliquer sur nouvelle zone

Entrer le nom de la zone
