Quota FSRM

Installation du rôle

Dans le gestionnaire de serveur, cliquer sur Gérer puis sur Ajouter des rôles et fonctionnalités.

Dans les rôles, cliquer sur la petite flèche devant Services de fichiers et de stockage puis faire la même chose pour Services de fichiers et iSCSI puis cocher la case Gestionnaire des ressources du serveur de fichiers.

Cliquer sur suivant, puis finir en cliquant sur installer.

Configuration du quota

Dans le gestionnaire de serveur, cliquer sur Outils puis sur Gestionnaire de ressources du serveur de fichiers.

Dans le gestionnaire de ressources, faire un double clic sur Gestion de quota, puis sur Quotas.

Dans le panneau de droite, cliquer sur Créer un qutoa..

Dans la fenêtre de création, spécifier le chemin d'accès pour sélectionner le volume sur lequel on souhaite mettre un quota.

Sélectionner une option déjà définie, ou choisir Définir des propriétés de quota personnalisées pour créer vos propres limites.

Cliquer ensuite sur Propriétés personnalisées… et ajouter les propriétés que l'on souhaite.

Cliquer sur OK puis sur Créer. Choisir si l'on souhaite enregistrer ou non le quota.
