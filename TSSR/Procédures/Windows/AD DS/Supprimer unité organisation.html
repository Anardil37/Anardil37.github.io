Supprimer unité organisation

Dans le gestionnaire de serveur, cliquer sur Outils puis sur Domaines et approbations Active Directory

Dans la nouvelle fenêtre, clic droit sur le domaine, puis cliquer sur Gérer

Dans le menu Affichage, il faut que Fonctionnalités avancées soit coché.

Faire un clic droit sur l'unité d'organisation que l'on souhaite supprimer puis cliquer sur propriétés.

Aller dans l'onglet Objet puis décocher la case Protéger l'objet des suppressions accidentelles

Cliquer sur Appliquer puis OK.

Refaire un clic droit sur l'unité d'oganisation puis cliquer sur Supprimer.
