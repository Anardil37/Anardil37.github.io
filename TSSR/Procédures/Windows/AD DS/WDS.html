Installation

Install-WindowsFeature -Name WDS -IncludeManagementTools

 

Dans le menu démarrer, aller dans Outils d'administration Windows puis cliquer sur Services de déploiement Windows

 

Faire un clic droit sur le serveur puis cliquer sur configurer le serveur
