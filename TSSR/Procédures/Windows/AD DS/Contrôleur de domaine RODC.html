Contrôleur de domaine RODC

Ajouter le contrôleur dans le domaine, puis dans le gestionnaire de serveur installer le rôle AD DS.

Sélectionner Ajouter un contrôleur de domaine à un domaine existant, puis entrer le domaine dans lequel on souhaite ajouter le contrôleur.

Cliquer sur Suivant.

Cocher la case Contrôleur de domaine en lecture seule.

Terminer l'installation.
