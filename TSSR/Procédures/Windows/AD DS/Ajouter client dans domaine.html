Ajouter client dans domaine

Dans le panneau de gauche du gestionnaire de fichiers, faire un clic droit sur Ce PC puis cliquer sur Propriétés

Dans la fenêtre qui s'ouvre, descendre jusqu'en bas, puis cliquer sur Rennomer ce PC (avancé)

Cliquer sur Modifier

Entrer le nom de la machine puis sélectionner Membre d'un domaine et entrer le nom du domaine.

Cliquer sur OK puis Appliquer
