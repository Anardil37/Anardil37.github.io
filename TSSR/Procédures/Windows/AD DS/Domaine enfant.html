Domaine enfant

Mettre le contrôleur de domaine dans un domaine parent

Lors de l'installation du rôle Active Directory, Ne pas sélectionner Ajouter à une nouvelle forêt mais choisir Ajouter un nouveau domaine à une forêt existante.

Choisir le type de domaine (domaine enfant) puis cliquer sur Sélectionner. Entrer les identifiants de l'administrateur du domaine parent puis sélectionner dans la liste le domaine parent que l'on souhaite.

Entrer le nom du nouveau domaine.

Décocher la case Créer une délégation DNS !!!
