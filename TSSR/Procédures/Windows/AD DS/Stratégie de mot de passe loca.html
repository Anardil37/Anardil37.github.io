Stratégie de mot de passe locale

Dans le gestionnaire de serveur, cliquer sur Outils puis sur Centre d'administration Active Directory.

Dans la nouvelle fenêtre, cliquer sur le domaine dans le panneau de gauche puis chercher Système. Double clic dessus pour l'ouvrir.

Faire un double clic sur Password Settings Container.

Faire un clic droit, puis Nouveau et Paramètres de mot de passe.

Ajouter les différentes règles que l'on souhaite pour les mots de passe.

Cliquer sur OK pour valider.
