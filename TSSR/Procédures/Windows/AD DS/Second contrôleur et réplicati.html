Second contrôleur et réplication

Ajouter le contrôleur dans le domaine, puis dans le gestionnaire de serveur installer le rôle AD DS.

Sélectionner Ajouter un contrôleur de domaine à un domaine existant, puis entrer le domaine dans lequel on souhaite ajouter le contrôleur.

Cliquer sur Suivant.

Choisir le site sur lequel se trouve le contrôleur de domaine.

Terminer l'installation.

Dans le gestionnaire de serveur, aller dans outils > Sites et services Active Directory

Déplacer les contrôleurs dans les différents sites.

Clic droit sur Inter-Site-Transports > IP puis cliquer sur Lien vers un nouveau site …

Donner un nom au lien puis sélectionner les sites que l'on souhaite inclure dans celui-ci.

Double clic sur la relation.  Modifier la valeur de réplication toutes les …

Cliquer sur Modifier la planification.. Puis, sélectionner les horaires pour les quelles la planification est possible ou non.
