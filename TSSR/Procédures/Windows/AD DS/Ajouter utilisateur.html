Ajouter utilisateur

Dans le gestionnaire de serveur, cliquer sur Outils, puis cliquer sur Domaines et approbations Active Directory

Faire un clic droit sur le domaine puis cliquer sur Gérer

Faire un clic droit sur l'unité d'organisation dans laquelle on souhaite ajouter un utilisateur, puis cliquer sur Nouveau puis Utilisateur

Entrer le Nom et Prénom de l'utilisateur, ainsi que son identifiant de connexion. Cliquer sur suivant

Entrer le mot de passe par défaut de l'utilisateur puis sélectionner les options souhaitées. Cliquer sur suivant puis terminer.
