Ajouter une GPO

Créer la GPO

Dans le gestionnaire de serveur, cliquer sur Outils puis sur Gestion des stratégies de groupe.

Dans la nouvelle fenêtre, faire un clic droit sur Objets de stratégie de groupe puis cliquer sur Nouveau…

Modifier les paramètres de la GPO

Dans le gestionnaire de stratégie de groupe, clic droit sur la stratégie, puis cliquer sur Modifier…

Séléctionner le paramètres que l'on souhaite modifier. Faire un double clic dessus pour ouvrir une nouvelle fenêtre.

Dans cette nouvelle fenêtre, on peut choisir la valeur que l'on souhaite donner à ce paramètre. On peut également trouver une aide qui explique ce que fait le paramètre, ainsi que les versions de windows sur les quelles le paramètre fonctionne.

Cliquer sur Appliquer puis OK.

Cliquer sur OK pour valider.

Lier la GPO

Faire un clic droit sur l'OU et cliquer sur Lier un objet de stratégie de groupe existant...

Sélectionner la GPO que l'on souhaite lier à l'OU puis cliquer sur OK.

La GPO est maintenant liée à l'OU.

Appliquer les GPO → gpupdate /force

Afficher les GPO → gpresul /R
