Agent Linux

Il faut récupérer le fichier veeam-release-deb_1.0.8_amd64.deb

On utilise ce fichier pour ajouter des sources:

dpkg -i veeam-release-deb_1.0.8_amd64.deb && apt update

 

On installe les paquets puis on redémarre

apt install -y veeam blksnap

reboot

 

 

L'agent est maintenant installé, pour y accéder, on utilise la commande:

veeam

 

On accepte les termes de la licence

On choisit Next puis Finish sans rien changer sur les pages suivantes
