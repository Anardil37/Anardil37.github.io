Depuis le serveur

On clique sur Backup Job puis sur Windows computer …

 

On choisit Server et Managed by backup server

 

On choisit le nom de notre tâche

 

On clique sur Add puis Individual Computer et on entre l'adresse IP de la machine

 

On choisit ce que l'on veut sauvegarder

 

On choisit le répertoire de backup dans lequel on souhaite placer notre sauvegarde

 

Ne pas cocher la case pour effectuer la sauvegarde une seule fois

 

On vérifie que tout est bon et on coche la case pour lancer la tâche
